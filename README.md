# VLESS Telegram Bot

Телеграм-бот для автоматического создания пользователей VLESS в панели 3X-UI.

## Возможности

- Запуск аккаунтов через Telegram.
- Модерация заявок администратором.
- Генерация VLESS ссылок (поддержка Reality и TLS).
- Интеграция с базой данных 3X-UI.

## Требования

- Python 3.10+
- Установленная панель 3X-UI.

## Быстрая настройка

Клонируйте репозиторий:

```bash
git clone https://github.com/arni30rus/tgbot-vless.git
cd tgbot-vless
```

### Установка через скрипт install (рекомендуемый способ)

Запустите установщик:

```sh
sudo ./install.sh
```

### Установка через Docker

Клонируйте репозиторий, переименуйте .env.examle в .env и отредактируйте файл.

### Запуск контейнера 
```sh
docker compose up -d --build
```

### Настройка переменных окружения

В обоих случаях нужно только заполнить файл .env. Откройте его и впишите свои данные:

    BOT_TOKEN — токен от @BotFather.

    ADMINS_ID — ваш Telegram ID.

    XUI_BASE_URL, XUI_USERNAME, XUI_PASSWORD — данные доступа к панели.

    И др.

### Запуск вручную (при установке скриптом)
```sh
source venv/bin/activate
python main.py
```

## Установка как сервис, при установке скриптом (Автозапуск) 

1. Отредактируйте файл tgbot.service:
    Замените /path/to/project на реальный путь к папке с ботом (например, /etc/tgbot-vless).

2. Скопируйте файл в систему:

```sh
sudo cp tgbot.service /etc/systemd/system/tgbot.service
```

3. Запустите сервис:

```sh
sudo systemctl daemon-reload
sudo systemctl enable tgbot.service
sudo systemctl start tgbot.service
```

## Особенности работы:

    Пользователь пишет боту /request_account.
    Админ одобряет заявку.
    Бот создает клиента в панели 3X-UI.

    ВАЖНО: После создания нужно зайти в веб-интерфейс 3X-UI, нажать кнопку "Редактировать Клиента".
    Затем ничего не меняя, нажать "Сохранить". После этого подключение активируется.

P.S. Обойти лишнее сохранение пока не удалось, т.к. API у 3x-ui отвечает на GET запрос созданного конфига пустой строкой.

Добавлена команда /reset_xray Перезагружает весь 3x-ui. На некоторых версиях 3x-ui перезагрузка помогает с тем, что не надо заходить в веб и нажимать "Сохранить".
